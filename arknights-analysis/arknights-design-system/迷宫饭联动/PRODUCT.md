# Product

## Register

brand

## Users

面向《明日方舟》玩家中**对日漫 IP《迷宫饭》(Delicious in Dungeon / ダンジョン飯) 有情感投入**的群体——尤其是看过九井谅子原作漫画或 Trigger 制作的 2024 年 TV 动画的玩家。次级受众是仅熟悉方舟、被"在地下城里炖魔物"这一独特 RPG 设定吸引而进入的玩家。

他们的使用情境通常是：

- 在动画播出窗口前后看到联动 PV、PR 物料
- 在游戏内点开"迷宫饭"联动活动入口
- 通关战斗关卡获得"魔物食材"，再到食谱界面合成菜肴 (典型的 cooking minigame)
- 获取限时干员玛露西尔 (Marcille)、莱欧斯 (Laios) 等
- 在分享物料里看到"博士 (玩家本人) 也加入了食堂队伍"的二次创作叙事

他们要完成的事是：把方舟世界的"塔防战术"嵌进迷宫饭 IP 标志性的"探险 + 烹饪"循环里，体会"一边打怪一边查食谱手册"的 RPG 桌游感。

## Product Purpose

「《明日方舟》× 《迷宫饭》联动」是配合 2024 年迷宫饭动画播出窗口推出的限定联动活动。它要做的两件事：

1. **复刻原作的"探险 + 烹饪"双循环**——让玩家在方舟塔防体系里仍能体验到"打完一关 → 翻开食谱 → 把刚获取的怪物食材做成菜"这套迷宫饭核心节奏。
2. **把方舟的干员档案体系翻译成"冒险者羊皮卷"**——用迷宫饭世界观的视觉语言重新包装"新干员介绍"这一方舟既有 UI 形制。

成功的标准是：玩家在第一次打开食谱界面时**立刻被"打开了一本旧 RPG 烹饪手册"的氛围击中**，在第一次查看玛露西尔档案时**立刻被"翻开了一卷探险者羊皮地图"的氛围击中**——前者是迷宫饭的灵魂，后者是方舟形制对该灵魂的尊重。

## Brand Personality

三个词：**羊皮卷、烹饪手账、桌游怀旧** (parchment, recipe diary, tabletop nostalgia)。

声音 (Voice)：像桌游夜里的一位 GM (game master) 翻开手册念出菜谱条目——半正式、半玩闹，自带一种"这是真的探险家在记录"的可信度。

调性 (Tone)：暖色 + 低饱和度 + 手绘笔触 + 纸张磨损质感。即便在最戏剧性的"爆炒源石虫"瞬间，画面整体仍保持"羊皮卷上一道菜的图鉴" 的克制感——**不是炫目食物摄影，是手绘食谱**。

情绪目标：玩家在食谱卡前感受到**翻动旧手账的安静喜悦**，在干员档案前感受到**展开一卷地图的探险期待**——两种感受共同支撑"在桌游夜里玩 D&D"的整体怀旧氛围。

## Anti-references

- **食物摄影模板 + 高光油亮 PBR 渲染。** 现代手游烹饪类内容的视觉默认值。它追求"诱人吃下去"，但本活动追求"诱人翻下去 (翻页)"——前者是商业广告，后者是手账。
- **米哈游"原神料理"式高对比度发光蒸汽 + 角色聚光灯演出。** 该路线把烹饪推向"演出秀"，与本活动追求的"手抄食谱"互斥。
- **国内手游联动惯用的"两个 LOGO 拼一起 + 渐变背景 + 巨大 CROSSOVER 字样"懒惰模板。** 与罗小黑联动同样的反例——本活动同样要求双 IP 平等署名 (Arknights × Delicious in Dungeon)。
- **赛博 / 霓虹 / 工业警戒色。** 方舟自身的全局调色板在本活动里被刻意压制——羊皮卷上没有警戒条纹。
- **过度卡通化的低龄烹饪游戏画风 (高饱和糖果色 + 圆头几何字)。** 迷宫饭原作的画风是有粗砺笔触的成人向漫画，不是儿童动画——这条调性必须守住。
- **现代极简餐厅风 (大留白 + 无衬线 + 白底冷色)。** 它表达"高级餐饮"，本活动表达"地下城烹饪"，二者完全相反。

## Design Principles

1. **羊皮卷是底层语言 (parchment is the substrate)。** 所有承载内容的"画布"——食谱卡、干员档案、剧情入口——都必须是羊皮卷或纸张材质，带可见的边缘磨损、纸纹噪点、轻微染色。**禁止纯色塑料底**。
2. **食材必须可读 (ingredients must be legible)。** 食谱卡遵循"成品图 + 配料图 + 配料名"的三段式不变结构——这是迷宫饭的图鉴语法。即便菜品视觉再夸张，配料展示永远清晰、对齐、可被一眼数清。
3. **手绘 > 渲染 (hand-drawn beats rendered)。** 所有食物 / 食材 / 道具插画必须是手绘风 sticker——可见的描边、轻微抖动的轮廓、面化处理的高光 (faceted highlight)。**禁止 3D 渲染或写实摄影**。
4. **双 IP 平等 (mutual attribution by mark)。** 沿用罗小黑联动原则——「明日方舟 ARKNIGHTS」与「迷宫饭 DELICIOUS IN DUNGEON / ダンジョン飯」两个完整字标必须同等级别共存，× 符号相连。
5. **桌游夜的字怀 (the typography of game night)。** 干员档案标题走衬线 italic display (Cormorant / Trajan italic)——这是 D&D 角色卡的视觉惯例。食谱标题走 Source Han Sans Heavy 黑体——这是日漫食谱书的视觉惯例。两套字怀共存而不互换。

## Accessibility & Inclusion

- **WCAG**：羊皮卷底色 (cream parchment) 与暗墨色字怀的组合天然达到 AA 级，但**烹饪 UI 中"配料 + 数量"信息密度高**——必须确保 ≥4.5:1 文本对比度，且数字字号不低于 14px。
- **色觉**：本活动主导色为暖色 + 中绿 + 红，对常见色觉差异友好；但**禁止仅靠"红色 = 失败 / 绿色 = 成功"传达必读信息**——烹饪结果反馈需带文本 + 图标双通道 (e.g. ✓/✕ 图标 + 文字)。
- **食物表达**：食谱内容涉及虚构怪物食材 (源石虫、角兽肉等)，部分插画风格可能包含粗砺幽默元素；推送物料应避免在"食"语义上做过于真实的肉感渲染，以免触发饮食敏感用户。
- **跨语言**：「迷宫饭」(*Delicious in Dungeon* 英版 / *ダンジョン飯* 日版) IP 字标在不同地区版本必须使用对应语种的官方字标，**禁止用机翻或自制字代替**。
- **减弱动态**：食谱合成动画 (锅冒烟、星点闪烁) 需提供 `prefers-reduced-motion` 静态版本——尤其是炒菜火光的高频闪烁。
- **桌游 / 漫画文化门槛**：迷宫饭的部分梗 (例如"魔物烹饪"对 D&D 派系的玩梗) 对没接触过桌游的玩家不可读——剧情入口需提供轻量的"你不需要懂 D&D 也能玩"的引导文案。
