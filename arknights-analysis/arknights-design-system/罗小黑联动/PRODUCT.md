# Product

## Register

brand

## Users

面向《明日方舟》玩家中**对中国原创动画 IP 有情感投入**的群体，特别是看过 *罗小黑战记* 番剧 (TV) / 大电影 (2019) 的玩家——他们与 IP 之间有现成的情感锚点，本活动是把这种锚点迁移到方舟世界观里。次级受众是仅熟悉方舟、被 IP 风格 (国漫水墨 + 萌系黑猫) 吸引而进入的玩家。

他们的使用情境通常是：

- 在 B 站 / 微博看到联动 PV、PR 物料 (大量出现于 2022 年 9 月)
- 在游戏内点开"故事集 *好久不见*"活动入口
- 通过通关活动关卡获取限时干员 (罗小黑、风笛、空) 与限时家具 (店铺、龙门小馆装饰)
- 在罗德岛 / 龙门小馆这类温馨场景里翻看角色档案与剧情对话

他们要完成的事是：在原本以**反乌托邦科幻 + 战术塔防**为底色的方舟里找到一种**温柔的喘息间隙**——和罗小黑、铃兰、风笛一起在春天的山林里走一程。

## Product Purpose

「《明日方舟》× 《罗小黑战记》联动 — 故事集"好久不见"」是 2022 年 9 月开启的限定联动活动。它要做的事是把两个差异巨大的 IP 接合在同一个剧情语境里：

- **方舟**：未来废土 + 工业警戒色 + 严肃军事风
- **罗小黑**：山林日常 + 春绿水墨 + 萌系国漫风

成功的标准是：玩家不会觉得罗小黑"被强行拉进方舟"，也不会觉得方舟"为了讨好 IP 而变软"。两边都尊重对方的本来面目，靠**春天的山林**这一共同的视觉与情绪底色把两个世界缝合到一起。"好久不见"四个字承担了情感钩子——它既是对 IP 老粉的招呼，也是对方舟玩家进入新故事时的迎接。

## Brand Personality

三个词：**春日、亲和、轻盈** (vernal, friendly, weightless)。

声音 (Voice)：像久别重逢的朋友在山道上打的招呼——熟悉、放松、不需要客套。

调性 (Tone)：高明度、暖偏冷的春绿主导，搭配水墨灰阶作为"出处感"的支撑。即便是萌系贴纸式的角色卡通风，也保留了水墨 + 印章的中式根基。整套活动**不该有一刻感觉冷峻或咄咄逼人**，但也不能滑向纯儿童化。

情绪目标：玩家在主 KV 与 PR 物料前感受到**春风扑面**，在干员档案卡前感受到**翻阅画册的安静**——两种感受在同一活动里前后呼应。

## Anti-references

- **国内联动手游"两个 LOGO 拼一起 + 紫红渐变背景 + 巨大 'CROSSOVER' 字样"模板。** 该模板是合作公关的视觉懒惰，与本活动追求的"两个 IP 都被尊重"诉求直接冲突。
- **方舟自身的工业警戒色 (industrial alert) 与赛博废土美学。** 这是被刻意压制的——本活动里几乎不出现霓虹橙、警戒条纹、工业网点，因为它们与"春日山林"互斥。
- **米哈游式联动惯用的"高饱和金属渐变 + 光晕粒子"。** 该路线追求的"奢华联动感"会让罗小黑的山野气立刻变质。
- **过度幼龄化的儿童海报色板 (高饱和糖果色 + 圆头几何字)。** 罗小黑是温柔，不是儿童；本活动的目标受众是青年到中年，不是低幼。
- **怀黍离式肃穆水墨。** 同 IP 内的另一种风格，但其悲剧底色不适合"久别重逢"的语境。

## Design Principles

1. **春绿是关节，不是底色 (lime green is a hinge, not a wash)。** 罗小黑的标志性春绿 (LXH lime) 不该铺满整张画布——它只在 (a) 联动专属 banner、(b) 春日页面的边缘装饰、(c) 角色服饰高光等"关节位置"出现。让春绿作为"亮起的一束光"而不是"涂满的一面墙"。
2. **方舟出形，罗小黑出气 (Arknights gives the form, Luo Xiaohei gives the air)。** 联动活动卡片仍沿用方舟既有的"代号 / 种族 / 出身 / 专精"档案式布局——这是方舟的形。但卡片的纸感、墨气、印章、笔触都向罗小黑的水墨风靠拢——这是罗小黑的气。两者各占一半。
3. **可爱要有出处 (cute with provenance)。** 萌系贴纸 (chibi sticker) 风格只允许在公关 banner 与社交分享物料里出现，并且必须让水墨 / 印章 / 山林等"中式国漫"底层元素同时在场。**禁止裸贴纸**——单纯靠"贴纸 + 大白边"会立刻沦为低龄派对。
4. **"好久不见"是文案锚 (the welcome line carries everything)。** 整个活动的情感基调由"好久不见"四字定下——所有副本名、奖励文案、对话语气都要回到这四字的"重逢叙事"上对齐。激烈、对抗、悬念性的措辞都要被压回。
5. **联动尊重各自署名 (mutual attribution by typography)。** 主 KV 与 PR 物料必须**同时**呈现「明日方舟 ARKNIGHTS」与「罗小黑战记 THE LEGEND OF LUOXIAOHEI」两个完整字标，并以一个细的 ×（cross 符号）相连。任何一方被缩小、模糊化或省略都视为违规。

## Accessibility & Inclusion

- **WCAG**：主调春绿 + 浅色纸面的组合天然存在文本可读性挑战。**所有功能性 UI 文本** (奖励名、活动倒计时、关卡名) 必须达到 AA 级 (≥4.5:1)，常见做法：在春绿底块上用 ink-black 文本而非白色文本。
- **色觉**：春绿对一类色弱用户区分度尚可，但对二色觉用户与 ink-wash-gray 区分度可能不足；**禁止仅靠"绿/灰差异"传达必读功能信息**——必读信息走形状 + 文本双通道。
- **减弱动态**：PV 中的山风掠过、墨点晕开等动效在游戏内 UI 复用时需提供 `prefers-reduced-motion` 静态版本。
- **听觉**：罗小黑大电影 BGM 元素若被复用作活动主题音乐，需在游戏设置中允许独立调节。
- **跨语言**：「罗小黑战记」(*The Legend of Luoxiaohei*) 与「明日方舟」(*Arknights*) 双 IP 字标在海外版本必须呈现拉丁文版本，且字号比与中文版本一致——这是 IP 联动的署名公平性。
- **儿童友好与年龄定位**：本活动情绪偏温柔但**不是儿童产品**；推送物料应避免使用 emoji 堆砌或儿童化插画风格的二次创作，以免误导分级与定位。
