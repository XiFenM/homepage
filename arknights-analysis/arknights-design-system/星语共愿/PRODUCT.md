# Product

## Register

brand

## Users

面向《明日方舟》的**老玩家与情感型核心用户**——尤其是从 2019 年首测 / 公测期就陪伴游戏走来的玩家，以及在剧情中投入了大量情感的"罗德岛家人型"玩家。他们的使用情境通常是：

- 在周年纪念时间窗 (anniversary window) 收到推送通知后第一时间打开游戏
- 在主页看到大幅周年 KV 与回顾性视频
- 在社交媒体分享自己的"罗德岛年终账单" (yearly stats card) 与生贺图
- 在游戏内领取限定道具、阅读纪念剧情

他们要完成的事不是"完成一个任务"，而是**确认"自己仍被记得"** ——希望从视觉、文案、奖励到剧情都能传递出"游戏团队和角色都还认得你"这层情绪。

## Product Purpose

「星语共愿」(Wishes from Stars) 是《明日方舟》2025–2026 跨年的**周年纪念主题活动** (anniversary special)，配合特别纪念 PV、生日特制内容 (博士特制蛋糕等限时玩法)、以及阶段性的 PV 播放量解锁奖励机制。

它要做的事是把"周年"这个相对抽象的时间节点，翻译成两件具体的视觉感受：
1. **仰望星空**——回望走过的年份，许愿，承担情感重量
2. **被庆祝**——博士被罗德岛同伴们送上一份属于"自己"的生日礼

成功的标准是：玩家在看到主 KV 时**先被一种宁静的浪漫击中**，然后在进入游戏后被生日蛋糕、彩带、博士专属道具等暖意触动；前者用作情感入口，后者用作情感落点。两者必须在同一活动里共存而不撕裂。

## Brand Personality

三个词：**澄澈、温柔、许愿** (luminous, tender, wishful)。

声音 (Voice)：像深夜天文台里轻声说出的祝福——既正式 (因为是一年一度)，又私密 (因为是写给"你")。

调性 (Tone)：高明度、低对比度、情绪走 "soft glow" 而非 "high contrast"。即便在最热闹的生日蛋糕画面里，也保持一种 pastel + 留白的克制。整套活动**不该有一刻感觉嘈杂**。

情绪目标：玩家在主 KV 与 PV 入口被"安静地打动"，在生贺玩法里被"轻盈地逗笑"。两个情绪点之间靠一条"星空—梦境—礼物"的隐喻链条贯穿。

## Anti-references

- **国内手游周年常用的"金红霓虹 + 周年大字 + 烟花"模板。** 它的喜庆度过高、信息密度过满，与本活动追求的安静浪漫直接冲突。
- **米哈游"星穹铁道"式金属质感 + 玻璃粒子 + 高饱和角色聚光灯。** 该路线的主旨是"璀璨"，本活动的主旨是"许愿"——许愿是低声的，璀璨是高声的。
- **"渐变彩虹 + 闪光颗粒 + 立体气球"的儿童派对模板。** 它确实属于"庆祝"语义，但缺乏情感重量，会让活动情绪滑向廉价。
- **"高对比度黑底 + 紫蓝霓虹 + 玻璃拟态 (glassmorphism)"的赛博周年模板。** 与方舟全局工业警戒色相近但完全错位——本活动反而需要远离工业感、走向梦境感。
- **怀黍离式水墨克制路线。** 这是同 IP 内的另一种风格，但其肃穆感不适合周年庆，会让玩家误以为剧情发生了悲剧。

## Design Principles

1. **澄澈而非闪亮 (luminous, not glittery)。** 用大面积柔光、留白和半透明层次表达"光"，而不是用金属高光、粒子闪点、镜面反射。前者是"星空"，后者是"舞台"。
2. **同活动两个调式 (two voices, one stage)。** 主调 (Star Mode) 走珍珠蓝 + 冰白 + 银，承担正式纪念；副调 (Celebration Mode) 走粉蓝奶油糖果色 + 手绘蛋糕，承担生贺玩法。两套调式必须在主活动入口、官方公告、PV 等情感锚点上**保持主调主导**；副调只在"博士生日"这条具体支线里被允许。
3. **手写感是必要的 (handwriting matters)。** 主标"星语共愿"的字怀和副标"WISHES FROM STARS"的草书拉丁体都必须保留可见的笔触感与轻微不规则——它表达的是"这是一份写给你的字条"。**禁止用机械几何字怀**。
4. **奖励即记忆 (rewards as memorabilia)。** 生贺道具 (蛋糕、车轮模型、礼盒) 在 UI 中应当被当作"被珍藏的小物件"呈现——加阴影、加描边、加贴纸感，使它们看起来值得被收藏，而不是任意的功能图标。
5. **博士是主角 (the Doctor is the centerpiece, not the operators)。** 周年 KV 的视觉重心应当让玩家感觉到"罗德岛在为你庆祝"。即便阿米娅 (Amiya) 等代表性干员出现，他们的姿态也是"望向你"或"为你举杯"，而不是"代你出战"。

## Accessibility & Inclusion

- **WCAG**：主调的高明度低对比度路线对正常视力玩家很美，但对低视力或屏幕反光环境用户构成可读性挑战。**所有功能性 UI 文本** (奖励名称、领取按钮、活动倒计时) 必须额外加深底板，确保 ≥4.5:1 对比度。在 KV 上叠加文本时，保留一层不低于 60% 不透明度的 deep-navy 局部底纹。
- **色觉**：粉蓝糖果色调中的 candy-pink 与 candy-blue 对二色觉用户区分度尚可，但禁止**仅靠粉蓝差异传达必读功能信息** (例如"红色按钮领取 / 蓝色按钮放弃")，必读信息需用形状 + 文本双通道。
- **减弱动态**：PV 中的星辰流转、彩带飘落、蛋糕展开动画在游戏内 UI 复用时需提供 `prefers-reduced-motion` 静态版本——尤其是粒子持续闪烁，对前庭敏感用户不友好。
- **听觉**：周年 PV 中的弦乐与角色独白承担了大量情绪传达，须配中英双语字幕，并在游戏内回放入口允许独立调节 BGM 与人声音轨。
- **情感安全**：纪念活动天然召唤"流逝感"，文案应当避免使用"永远""唯一一次""不再"等绝对化承诺词，以免触发因日常账号疏远而产生的情绪反弹。
