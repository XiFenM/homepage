# V3 · 羊皮→星空过场 (Parchment-to-Cosmos Transition)

> **Wave 5** · 工具 = seedance2 (`bytedance/doubao-seedance-2.0`) · 起始帧 = `static/S6/final.png`

## 1. 用途与规格 (Spec)

| 项 | 值 |
|---|---|
| 用途 | 探索站点击「观星塔」hotspot 后的"世界切换"过场 — 从地图视角进入观星塔上方星空 |
| 起始帧 | [S6 异世界地图 final.png](../../generated/static/S6/final.png) |
| 输出格式 | MP4 |
| 分辨率 | 720p (16:9) |
| 时长 | 5 秒（API 最短） |
| 价值 | **Wave 5 中最高价值** — 决定探索站进观星塔的"戏剧性"上限 |
| 备选方案 | CSS 600ms opacity-blend（DESIGN.md §5 原 spec） |

## 2. 主 Prompt

见 [`generator/prompts.py`](../../generator/prompts.py) 的 `V3`。核心要求：
- 起初镜头看到完整的羊皮卷地图（保持 1–1.5 秒不变）
- 接着镜头**缓慢向上抬升 + 略微推近观星塔山顶**，地图从画面下方滑出
- 过渡到**深邃夜空** OKLCH 15% 0.03 260
- 末尾画面应是**全屏星空**，可见星点 + 银河带 OKLCH 72% 0.08 240
- 整个过渡是从"翻看地图"到"抬头仰望"的视觉转移
- 保持手绘油画风格
- **禁止**变成 3D / 写实摄影 / 科幻 CG / 出现文字

## 3. 验收清单

### 3.1 通用项

- [ ] 起始帧画面与 S6 final.png 高度一致
- [ ] 色调保持原画 OKLCH 色板（地图 + 观星塔冷光晕 + 星空深蓝紫）
- [ ] 仍是手绘油画风，不变 3D / 写实 / 科幻 CG

### 3.2 动效项

- [ ] 镜头**确实向上抬升**（不是横向平移、不是缩小）
- [ ] 观星塔山顶在过渡过程中始终在画面上半部
- [ ] 羊皮卷边缘的淡化是**渐进**的，不是硬切换
- [ ] 末尾画面真的呈现星空（可见星点 + 银河带），不是黑屏
- [ ] 过渡有清晰的"从大地到天空"的语义流动

### 3.3 静态保留项

- [ ] 没有意外文字 / 标注 / UI
- [ ] 没有意外的飞船 / 火箭 / 龙等"额外内容"

## 4. 风险与回退

| 失败模式 | 应对 |
|---|---|
| 模型不知道怎么从"地图"过渡到"星空"，生成混乱 | 把起始帧改为 S6 的观星塔局部裁剪 + 用文字描述更具体的镜头运动 |
| 末尾不出现星空 / 末尾仍是地图 | "the final 1–1.5 seconds MUST show a starfield, no parchment visible" |
| 镜头剧烈晃动 / 加速度过大 | "smooth slow camera tilt, like a long exhale" |
| 出现高饱和星云 | 反例段加 "no Hubble nebulae, no colorful gas clouds" |
| 出现文字 | "no text anywhere" |
