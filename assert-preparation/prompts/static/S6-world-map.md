# S6 · 异世界等距地图（探索站底图）(World Map)

> **Wave 3** · 工具 = gpt-image-2 · 关联组件 [DESIGN.md §5 World Map](../../../DESIGN.md)

## 1. 用途与规格 (Spec)

| 项 | 值 |
|---|---|
| 出现位置 | 探索站全屏底图，访客在它上面慢慢行走 |
| **本次输出** | **3840×2160 PNG**（4K UHD，16:9，API 最大边长 3840px 限制；DESIGN.md 原 spec 4096×2304 不支持，等比降级） |
| 后处理 | WebP 转码（80% 质量） + 准备 @1x / @2x → `public/art/map/world-map-base.webp` |
| Token 锚点 | `map-parchment` = `oklch(89% 0.04 80)` (底) · `map-parchment-deep` = `oklch(78% 0.05 75)` · `map-parchment-edge` = `oklch(68% 0.06 70)` · `map-burgundy` = `oklch(34% 0.12 25)` (建筑描线) · `map-dungeon-olive` = `oklch(42% 0.08 110)` (森林山林) · `map-mustard-gold` = `oklch(72% 0.14 75)` (hotspot/路径) · `cosmos-pearl-blue` = `oklch(72% 0.08 240)` (山顶冷调暗示) |
| 风险等级 | **高** — PLAN.md §9 列为最易翻车，预估 3–5 轮迭代 |

## 2. 主 Prompt (中文主版)

```
生成一张等距视角（约 45° 俯视斜视）的手绘异世界地图油画式数字插画，绘在一整张大羊皮纸上。整张地图覆盖一片小型虚构大陆，**大陆上分布着 5 个独立区域**，按以下方位精确排布于 16:9 横向画面：

【底层羊皮材质】
- 主底色 OKLCH 89% 0.04 80（淡黄牛皮纸调，温暖偏冷）
- 整张羊皮有 grain 颗粒纹（约 10% 强度）
- 翻动多次形成的深旧区，颜色 OKLCH 78% 0.05 75，零散分布
- **画面四角与四边的纸缘**有明显磨损、折痕、轻微焦边，颜色 OKLCH 68% 0.06 70
- 整张地图就是这张羊皮卷展开后的完整面，不要画羊皮卷的卷起部分（这是平铺展开版）

【五大区域——必须严格按方位绘制】

**1. 西部（画面左侧约 1/4 区域）「鸢尾平原」**
- 缓坡丘陵，覆盖紫色与淡蓝色的鸢尾花/薰衣草丛（密度约 30%）
- 一条蜿蜒的小路从平原贯穿，连接周边
- 散布 2–3 块路标（小石碑或木牌剪影，无文字）
- 整体色调：羊皮底色 + 淡淡的紫调（鸢尾花田的氛围色）

**2. 北方（画面左上区域，**比 5 号高峰矮一档**的低山）「钢铁书库」**
- 一座深色石塔，铁箍式建筑结构（塔身有水平铁带装饰），半嵌入低山中
- 塔身颜色 OKLCH 34% 0.12 25（沉酒红色描线 + 灰石身）
- 塔身窗口透出微弱暖光，暗示里面有书有人

**3. 山顶（画面正上方中央，**全图最高峰**）「观星塔」**
- 一座细长高耸的天文台，顶部带圆穹（dome），圆穹有一道窗口可见望远镜剪影
- **山顶区域的色调比其余地图略冷**——主色仍是羊皮底，但山岩与天空交界处带一丝 OKLCH 72% 0.08 240（cosmos-pearl-blue）的冷光晕，暗示"再往上就是星空"
- 此处是整张地图的"视觉锚点最高点"

**4. 东方（画面右侧约 1/4 区域）「烹饪窝棚」**
- 一座小木屋，屋顶冒着炊烟（烟柱细而温暖，OKLCH 75% 0.04 65）
- 屋外周围有香草园 / 蔬菜园（小色块拼合的植物纹理）
- 屋内通过门口透出暖光（OKLCH 82% 0.10 75 暖灯光）
- 旁边可能有一口大锅或柴堆细节

**5. 南方边缘（画面下边缘）「旅人客栈」**
- 一座朴素的两层木屋，窗口透出温暖光（OKLCH 82% 0.10 75）
- **客栈位置在画面下边缘，再往下地图渐渐淡出至雾气**——边界淡化，不是硬切
- 客栈附近有马厩剪影 / 通向北边的小路

【地形与连接】
- 5 个区域之间用 OKLCH 42% 0.08 110（map-dungeon-olive）的暗橄榄绿填涂森林、山林、丘陵
- 区域之间有蜿蜒的小路、小溪、桥梁连接，路径用 OKLCH 72% 0.14 75 的芥末金描线
- 山脉用斜笔触手绘（不是 3D），峰顶用淡淡 highlight，山阴用深一档绿
- 河流 / 溪流用淡淡的橄榄绿描线 + 浅蓝灰填涂

【Hotspot 标记】
画面上散布 **9–10 个金色发光小圆点**（OKLCH 72% 0.14 75 芥末金，每个直径约画面宽度的 1.2%–1.5%），位置精确分布如下：
- 鸢尾平原内 2 个（散布于平原中心与小路附近）
- 钢铁书库塔基附近 1 个
- 烹饪窝棚周围 2 个（屋前一个、屋侧一个）
- 观星塔山顶 1 个（**比其它略大一点，约画面宽度 1.8%**）
- 旅人客栈门口 1 个
- 区域之间的过渡地带 2–3 个（散布于森林路径、河流交叉点等地）

每个金色 hotspot 圆点应**亮但不刺眼**——像羊皮卷上的小蜡封印。**不要画 halo / 光晕** —— 后期 SVG 会叠加动效。

【绘画风格】
- 完全手绘羊皮笔记画风：可见笔触、深浅自然
- **不是数字地图、不是现代 infographic、不是 3D 渲染、不是 GIS 地图**
- 灵感参照：托尔金《魔戒》中土世界手绘地图、D&D 经典战役地图（A1 Slave Lords / Ravenloft）、《迷宫饭》"美味食谱书"插图、Studio Ghibli 在《天空之城》《幽灵公主》中的世界全景画
- 山林用斜笔小点状植被（不画详细单棵树）

【完全禁止】
- 任何文字、字母、数字、地名、坐标、刻度、罗盘度数
- 任何指南针、比例尺、图例
- 任何现代 UI 元素：按钮、图标、数据可视化框、渐变光晕
- 高饱和荧光色：地图整体应"温暖、低饱和、烛光下看到的旧地图"

【羊皮卷外缘】
- 画面四角有明显的羊皮卷折痕、磨损、轻微焦边
- 羊皮卷外的画面背景**透明**（用纯白替代抠透明）—— 这张图最终会上墙到探索站全屏，背景透明可让我们之后在底层加额外纹理或环境光

输出：3840×2160（4K UHD，16:9 横向），纯白背景，无任何边框装饰，地图本身呈不规则羊皮卷形态居中铺满约画面 92% 面积。
```

## 5. 验收清单 (Acceptance Checklist)

### 5.1 通用项

- [ ] 羊皮主底色 OKLCH 抽 3 点落在 89%±5% L、0.04±0.02 C、80±10° H
- [ ] 森林橄榄绿 OKLCH 抽 1 点落在 42%±5% L、0.08±0.02 C、110±10° H
- [ ] hotspot 金色 OKLCH 抽 1 点落在 72%±5% L、0.14±0.02 C、75±10° H
- [ ] 无 anti-reference 触发
- [ ] 整张地图无任何文字 / 字母 / 数字（gpt-image-2 这一项压力大，需严格核查）

### 5.2 五大区域识别项（核心）

- [ ] **鸢尾平原**（西/左 1/4）—— 鸢尾花田 + 蜿蜒小路 + 路标可识别
- [ ] **钢铁书库**（北/上中偏左）—— 铁箍石塔 + 嵌入低山 + 塔窗透光
- [ ] **观星塔**（顶/正上中央，最高峰）—— 圆穹天文台 + 望远镜剪影 + **山顶冷调（带 cosmos-pearl-blue 一丝痕）**
- [ ] **烹饪窝棚**（东/右 1/4）—— 木屋 + 炊烟 + 香草园 + 门口透暖光
- [ ] **旅人客栈**（南边缘，画面下边缘）—— 两层木屋 + 边界**渐渐淡出至雾气**
- [ ] 5 区方位与上述描述匹配，不串位

### 5.3 Hotspot 标记项

- [ ] 总数 **9–10 个**（少于 8 或多于 11 重生）
- [ ] 鸢尾平原内 2 个、钢铁书库 1 个、烹饪窝棚 2 个、观星塔 1 个（略大）、旅人客栈 1 个、区域间 2–3 个
- [ ] 每个金点亮但不刺眼，**无 halo / 光晕**（halo 由前端动效负责）
- [ ] 观星塔的金点比其它略大一点（约画面宽度 1.8% vs 其它 1.2–1.5%）

### 5.4 风格与材质

- [ ] **等距视角**（约 45° 俯视斜视），不是平视、不是正俯视
- [ ] 羊皮纸 grain 完整保留
- [ ] 四角与四边可见羊皮折痕 / 磨损
- [ ] 手绘笔触可见，不是矢量、不是 3D
- [ ] D&D / Tolkien / 《迷宫饭》笔意

## 6. 已知风险与回退

| 失败模式 | 应对 |
|---|---|
| 5 区识别度不足、串位 | **拆图方案**：分别生成 5 张 1024×1024 区域图（每张专注 1 个区域 + 周边地形），再 Photoshop 合成到等距底图上 |
| Hotspot 数量不对（少于 8 或多于 11） | prompt 加粗 "EXACTLY 9 OR 10 small gold dots, count them precisely"；若 AI 仍乱生成，hotspot 后期手动 SVG 标记 |
| 出现意外文字 | gpt-image-2 这是已知问题，prompt 顶部加 "ABSOLUTELY NO TEXT, LETTERS, NUMBERS, OR ANY WRITTEN MARKS ANYWHERE ON THIS MAP. PURELY VISUAL ILLUSTRATION." |
| 等距视角变平视 | "isometric 45° axonometric view, viewer looking down at angle, NOT flat top-down, NOT side-view" |
| 整体偏现代 / infographic 感 | 加粗"vintage hand-painted cartography, ink wash + watercolor on aged parchment, painted in the era before printing presses" |
| 颜色饱和度过高 | "muted earth tones, low saturation, dusty palette, like firelight on old parchment" |
| 4K 出图崩坏（细节糊） | 回退 2560×1440，再 PIL 上采样 |
