# S5 · 简约站邀请物件 (Quiet-Mode Object) — 三变体

> **Wave 2** · 工具 = gpt-image-2 · 关联组件 [DESIGN.md §5 Homepage Fork Scene](../../../DESIGN.md)

## 1. 设计史 (Design History)

**v1（已废弃）**：春绿便签（sticky note）。第一轮生成 4 张候选质量都很好，但访客评估认为**便签的语义太弱**——它说"附注/提醒"，不说"翻开看看我是谁"。与左侧 S4 羊皮卷"一整个世界等你走"的强语义不对等。

旧 sticky note 的生成数据已归档至 [`generated/static/_legacy-S5-sticky-note/`](../../generated/static/_legacy-S5-sticky-note/) 备查。

**v2（本卡）**：拆成三个候选物件（A/B/C），都满足"翻开/解开就能直接读到我"的强语义，与羊皮卷视觉对等。

## 2. 用途与共同规格 (Spec)

| 项 | 值 |
|---|---|
| 出现位置 | 首页 hero 桌面**右中**位置（与 S3 书桌、S4 羊皮卷合成在同一帧） |

## 3. 三变体说明

### S5A · 春绿布面精装笔记本（带丝带书签，闭合）
- 现代精装小本，春绿布/皮封面，闭合
- 一根稍深一档的暖橄榄绿丝带书签从书顶探出
- 封面空白；四角微磨损（暗示在用）
- **语义**：私人档案 / 个人记录 / 翻开就能直接读到的"我自己"
- **优势**：与羊皮卷视觉对等；春绿大面积承载；现代但不冰冷
- **风险**：现代精装与羊皮卷古朴感同台需要后期看实际效果

完整 prompt 见 [`generator/prompts.py`](../../generator/prompts.py) 的 `S5A`。

### S5B · 春绿布面线装书（中式订线）
- 中式线装本，春绿粗织布封面，左侧装订边可见 4–6 道缝线
- 封面右上角有空白白色书签纸（典型古籍标题位）
- 封面整体偏新但边角微磨损泛黄
- **语义**：东方档案 / 故人手札感
- **优势**：与怀黍离印章、晚樵庐字标的中式骨气对齐
- **风险**：偏古典；与简约站后续展示的现代博客内容可能违和

完整 prompt 见 `S5B`。

### S5C · 春绿丝带束扎的档案卡片堆（v2，已定稿）
- 5 张米白色档案卡片**微微扇形铺开**（每张右上角清晰可见）
- 每张右上角各有一个**微小手绘小标记**（呼应"我有几面"）：
  - 顶层：淡墨微缩山水（3–4 笔远山+小树）
  - 第 2 张：墨色单茎花草
  - 第 3 张：3 点构成的小三角星座
  - 第 4 张：朱砂红 4mm 小章（呼应站点 seal-red）
  - 最底张：空白（保留"未写就"的留白意味）
- 春绿丝带宽度提至画面 7%，横向束扎中部
- **小蝴蝶结**（贴卡 / 收紧 / 不蓬起）

> **v2 spec 演化注**：v2 prompt 原本要求"单结、不打蝴蝶结"（试图避开礼品感），但实际生成结果上**用户偏好留有小蝴蝶结**——它让卡片堆更有"被人主动整理过"的人情味，比硬冷的单结更契合"夜归桌上随手放的私人档案"。**最终 spec 接纳蝴蝶结**，仅约束其形态：贴卡、紧致、不蓬不翘、丝带尾不长飘。

**v1（已废弃）**：每张空白卡 + 蝴蝶结。问题：婚礼伴手礼/私信件捆扎的视觉模板太强；春绿丝带宽度仅 4% 权重弱；卡片间无差异不能传达"我有几面"。

完整 prompt 见 `S5C` (v2)。

## 4. 生成参数建议

每变体 4 张候选 × 3 变体 = 共 12 张。命令：

```bash
uv run python generate.py S5A S5B S5C
```

## 5. 验收清单（共用）

### 5.1 通用项

- [ ] 春绿色（封面/丝带）OKLCH 抽 3 点落在 86%±5% L、0.16±0.03 C、125±10° H
- [ ] **不偏黄绿（H > 140°）、不偏蓝绿（H < 110°）、不偏荧光（C > 0.20）、不偏苔绿（L < 70%）**
- [ ] 无 anti-reference 触发
- [ ] 背景纯白可抠透明

### 5.2 物品共用项

- [ ] 75° 俯视角，与 S3、S4 视角一致
- [ ] 阴影方向：朝右下方（光源左上）
- [ ] **无任何文字 / 字母 / 数字 / 印章 / 烫金 logo**
- [ ] 油画式手绘笔触可见，非照片、非 3D、非塑料

### 5.3 各变体专项

- **S5A 笔记本**：闭合状态、丝带书签从顶部探出 4cm 左右、四角微磨损、装订线/书脊弧线可见
- **S5B 线装书**：左侧 4–6 道缝线清晰、右上角空白白色书签纸位、纸边切口分层可见
- **S5C 卡片堆**：5–7 张层叠、最顶卡略偏移、丝带宽度约 4% 画宽、蝴蝶结小巧不喧宾、丝带尾垂 1–2cm

## 6. 已知风险与回退

通用问题与原 sticky note 卡片相同（春绿偏色、出现文字、塑料感），见 [`_legacy-S5-sticky-note/`](../../generated/static/_legacy-S5-sticky-note/) 备查。

变体特定风险：
- S5A 出现"商标 logo"——加强反例 "no embossed brand mark, no Moleskine logo"
- S5B 缝线被画成现代订书钉——强调 "traditional thread loops, not metal staples"
- S5C 蝴蝶结过于喜庆——强调 "small simple bow, not festive"
