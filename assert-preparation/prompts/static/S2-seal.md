# S2 · 印章红方章 (Signature Seal)

> **Wave 1** · 工具 = gpt-image-2 · 关联组件 [DESIGN.md §5 Signature Seal](../../../DESIGN.md)

## 1. 用途与规格 (Spec)

| 项 | 值 |
|---|---|
| 出现位置 | 全站每张内容卡 / 章节末尾右下角 / 语言切换标签活跃态 |
| 渲染尺寸 | 4mm × 4mm（约 16px on screen，等比缩放） |
| **本次输出** | 256×256 PNG ×2 张（中文版、拉丁版），**透明背景** |
| 后处理 | 直接入库 (`public/art/signatures/seal-jie.png`、`seal-xfm.png`) |
| Token 锚点 | `seal-red` = `oklch(48% 0.18 28)`（朱砂红，不带橘也不带紫） |

> **本卡片要求生成两张图**：A 版「桀」单字、B 版「XFM」三字母。两版尽量同一会话同一种子下连续生成，保证红色一致与边缘风化感同源。

## 2. 主 Prompt — A 版「桀」单字 (中文主版)

```
生成一张中国传统朱砂印章的扫描图：单个汉字「桀」，篆书（小篆为基础，略带秦汉
印章风格）。印章为正方形（4×4 比例），红色为传统朱砂红（OKLCH 48% 0.18 28），
不偏橘、不偏紫。

视觉特征——这是真人手按的章迹，不是数字贴纸：
- 边缘并不齐整，四角有细微的红色断裂或残缺，像盖压不均
- 字与四边之间的边框是红色实线，框线粗细 1–2mm，但有自然的浓淡变化
- 「桀」字以白底反白方式呈现（即字本身是留白，红色是底）
- 篆书结构正确：「桀」上部「舛」舞步状对称、下部「木」字稳立
- 画面有极轻的朱砂粉颗粒散布在章外 1–2mm 的范围（飞墨感），强度约 5%
- 整体有手按的轻微模糊或边角虚化，特别是右下角偏弱（按压不平）

输出：透明背景 PNG，256×256，仅印章本体居中（章占画面 90%，四周留 5% 透明缓冲），
无任何额外文字、签名、阴影或装饰元素。

风格参考：齐白石、吴昌硕个人落款印；故宫博物院藏明清私印；西泠印社印谱。

反例：NOT 数字矢量印章贴纸，NOT 矢量化干净的几何边缘，NOT Photoshop "stamp" 滤镜
效果，NOT 卡通风格，NOT 现代设计感的扁平 logo，NOT 3D 雕刻效果，NOT 红色 LED
发光，NOT 现代美工字体输入篆书生成器的产物。
```

## 3. 主 Prompt — B 版「XFM」三字母 (中文主版)

```
生成一张中国传统朱砂印章的扫描图：三个拉丁字母「XFM」横排，但**采用篆书化的
笔形处理**（笔画末端方折、转角圆中带方、每一笔有"刻刀痕"的微锐角）。印章为
正方形（4×4 比例），红色为传统朱砂红（OKLCH 48% 0.18 28），不偏橘、不偏紫。

视觉特征——这是真人手按的章迹，不是数字贴纸：
- 边缘并不齐整，四角有细微的红色断裂或残缺
- 字母与四边之间是红色实线边框，1–2mm 粗细，浓淡有自然变化
- 三字母以白底反白方式呈现（字母是留白，红色是底）
- 字母布局精确：X、F、M 三字等宽并排，间距匀称，整体居中略偏上
- 字母的笔形受篆刻刀法启发：垂直笔画上下端略尖、横画转角带方折，
  笔形粗细一致（约为字高 1/8），不是手写体也不是衬线体
- 画面有极轻的朱砂粉颗粒散布在章外 1–2mm 范围（飞墨感），强度约 5%
- 整体有手按的轻微模糊或边角虚化

输出：透明背景 PNG，256×256，仅印章本体居中（章占画面 90%），无任何额外文字、
签名、阴影或装饰元素。

风格参考：传统中国印章 + 西方现代刻字工艺融合；西泠印社"中西合璧"实验印；
20 世纪初日本"汉字+假名"印章。

反例：NOT 数字矢量印章贴纸，NOT 矢量化干净的几何边缘，NOT Photoshop "stamp"
滤镜效果，NOT 任何衬线 / 无衬线系统字体直接套印，NOT 卡通风格，NOT 现代 logo
扁平设计感，NOT 3D 雕刻效果。
```

## 6. 验收清单 (Acceptance Checklist)

### 6.1 通用项

- [ ] 朱砂红 OKLCH 抽 3 点落在 48%±5% L、0.18±0.02 C、28±10° H
- [ ] 不偏橘（H > 40°）、不偏紫（H < 15°）
- [ ] 透明背景干净，无残留白底 / 灰底

### 6.2 印章特定项（A、B 版共用）

- [ ] **正方形 4:4 比例**，章本体居中占画面 90%
- [ ] **边框存在**——四边可见红色实线边框，不是裸字
- [ ] **手按质感**——边缘有不齐 / 角部有缺失 / 红色浓淡有自然变化
- [ ] **不是矢量光滑** —— 边缘可见微毛刺、纸纤维痕迹
- [ ] **飞墨颗粒** —— 章外 1–2mm 有极淡的红色粉点散布（强度约 5%）
- [ ] **没有意外元素** —— 无背景纸纹（背景是透明的）、无落款外字、无水印

### 6.3 A 版「桀」字特定项

- [ ] **「桀」字结构正确**——上「舛」、下「木」，不缺笔不串字
- [ ] **篆书笔意**——笔画方中带圆，不是楷书也不是行书
- [ ] **白底反白**——字是空白、红色是底（不是红字白底）

### 6.4 B 版「XFM」字母特定项

- [ ] **X、F、M 三字母可识别**——不变形为符号
- [ ] **三字母等宽并排**——间距匀称，整体居中
- [ ] **篆刻化处理**——垂直笔画端部略尖、转角带方折
- [ ] **笔画粗细一致**——约为字高 1/8，无明显粗细对比
- [ ] **不是衬线 / 无衬线系统字体直接套印**

## 7. 已知风险与回退

参见 [PLAN.md §9](../../PLAN.md#9-风险与回退-risks--fallbacks)。

| 失败模式 | 应对 |
|---|---|
| A 版「桀」字篆书结构失败 | 查《说文解字》/ 西泠印社印谱中「桀」字篆形，作为参考图喂回 gpt-image-2 |
| B 版「XFM」字母被处理成衬线/手写 | 加强反例 + 提示词强调 "carved square block letters with seal-script edge treatment" |
| 红色偏橘 | 把 OKLCH 28° 提示改为 "traditional Chinese cinnabar red (朱砂红)，which is a deep slightly cool red, NOT orange-red, NOT scarlet" |
| 红色偏紫 | "NOT crimson, NOT magenta, NOT burgundy" |
| 完全失败（连续 3 轮 8 张全废） | 改用 Photoshop 手工合成：红色色块 + 篆书字典图扫描 + 飞墨 brush + 边缘 noise |
