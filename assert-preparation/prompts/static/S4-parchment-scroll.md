# S4 · 半展开羊皮卷地图（首页左物件）(Parchment Scroll)

> **Wave 2** · 工具 = gpt-image-2 · 关联组件 [DESIGN.md §5 Homepage Fork Scene](../../../DESIGN.md)

## 1. 用途与规格 (Spec)

| 项 | 值 |
|---|---|
| 出现位置 | 首页 hero 桌面**左中**位置（与 S3 书桌、S5 春绿便签合成在同一帧） |
| **本次输出** | **1024×768 PNG**（API 最小像素要求，原 spec 256×256 无法直生） |
| 后处理 | 抠透明背景（remove.bg / Photoshop） → `public/art/homepage/parchment-scroll.png` |
| Token 锚点 | `map-parchment` = `oklch(89% 0.04 80)` · `map-parchment-edge` = `oklch(68% 0.06 70)` · `map-mustard-gold` = `oklch(72% 0.14 75)` · `map-burgundy` = `oklch(34% 0.12 25)` · `map-dungeon-olive` = `oklch(42% 0.08 110)` |

## 2. 主 Prompt (中文主版)

```
生成一张俯视角的羊皮卷扫描图：一张半展开的旧羊皮卷地图，平铺在桌面上，从正上方看下去。羊皮卷整体呈横向的、略不规则的矩形——左右两端都向内卷起约 12%，中间约 76% 是平摊的、可以看到上面绘有内容的区域。

【羊皮卷材质】
- 主底色 OKLCH 89% 0.04 80（淡黄牛皮纸调，温暖偏冷）
- 表面有 grain 颗粒纹（约 10% 强度）和淡淡的水渍痕、折痕
- 翻动多次形成的深旧区，颜色 OKLCH 78% 0.05 75，零散分布于纸面
- 卷轴的两端卷起处和纸边磨损部位，颜色更深 OKLCH 68% 0.06 70
- **不要画羊皮卷下方的桌面或阴影**——这张图要透明背景出（仅羊皮卷本体）

【羊皮上的绘画内容——精确描述】
平摊的 76% 区域上，绘有一张异世界等距地图的**一角**（不是完整地图、只是一角、暗示更广阔的世界还在卷起部分里）：
- 上方约 1/3 处：一座**塔的剪影**（OKLCH 34% 0.12 25 沉酒红色描线），塔身细长、顶部带圆穹观星台，山岩的轮廓托着塔基
- 中部约 1/3 处：一条**蜿蜒的小溪**（OKLCH 42% 0.08 110 暗橄榄绿），从右上流向左下，沿途有几块石头
- 下方约 1/3 处：散布 **3–4 个金色小光点**（OKLCH 72% 0.14 75 芥末金），这些是地图上的 hotspot 标记，圆形、亮但不刺眼，每个直径约画面宽度 2%
- 山林、地形之间用淡淡的笔触填涂，整体感觉像 D&D 桌游团里手绘的战役地图

【绘画风格】
全部用手绘羊皮笔记画风：可见笔触、深浅自然、用淡墨/酒红色描线 + 暗绿色填涂地形 + 金色点缀 hotspot。**不是数字地图，不是现代 infographic，不是 3D 渲染**。

【构图与留白】
- 羊皮卷在画面中**居中**，占画面约 90% 面积
- 四周有约 5% 透明边缘（不要画背景）
- 两端的卷起处要画出明显的"翻折阴影"，体现纸张厚度

【完全禁止】
- 不要任何文字、字母、数字、地名、坐标、刻度、等高线数字
- 不要任何指南针、比例尺、图例
- 不要画下方的桌面、桌面阴影、其它物件

风格参考：托尔金《魔戒》中土世界手绘地图、D&D 团战役地图、《迷宫饭》中"美味食谱书"的手绘插图、星语共愿羊皮笺。

反例：NOT 数字地图，NOT 3D 渲染地图，NOT 卫星影像感，NOT 现代信息图（infographic），NOT 矢量光滑曲线地图，NOT 高饱和颜色，NOT 卡通游戏地图（如《动物森友会》《塞尔达》UI 风），NOT 任何文字标注。

输出：1024×768，纯白背景（后期会抠透明），仅羊皮卷本体居中，画面边缘留 5% 白边。
```

## 5. 验收清单 (Acceptance Checklist)

### 5.1 通用项

- [ ] 羊皮主底色 OKLCH 抽 3 点落在 89%±5% L、0.04±0.02 C、80±10° H
- [ ] hotspot 金点色 OKLCH 抽 1 点落在 72%±5% L、0.14±0.02 C、75±10° H
- [ ] 描线色 OKLCH 抽 1 点落在 34%±5% L 附近的酒红
- [ ] 无 anti-reference 触发

### 5.2 羊皮卷特定项

- [ ] **羊皮形态正确**——半展开、左右两端可见卷起、中间 ~76% 平摊
- [ ] **平摊区有完整地图内容**：塔（上）、小溪（中）、3–4 个金色 hotspot（下）
- [ ] **塔形清晰**——能识别塔身 + 圆穹顶（观星台）
- [ ] **小溪明确蜿蜒**——不是直线、不是模糊色块
- [ ] **3–4 个 hotspot 金点**——数量在 3–4 之间，不超过 5
- [ ] **无文字 / 字母 / 数字**——零容忍，任何位置出现文字立即重生
- [ ] **背景纯白，可抠透明**——羊皮卷外不能有桌面、阴影或其它元素
- [ ] **手绘笔触可见**——不是矢量、不是 3D
- [ ] **卷起处有翻折阴影**——体现纸张厚度，不是平面图层

## 6. 已知风险与回退

| 失败模式 | 应对 |
|---|---|
| 出现意外文字 / 字母 / 数字 | 重生（gpt-image-2 这是已知问题，prompt 加粗"NO text whatsoever"） |
| 地图细节过密导致塔/溪/点辨识不清 | 简化 prompt，降低描述细节，留更多笔触空间 |
| 颜色饱和度过高 | 加 "muted, dusty earth tones, low saturation" |
| 画风偏卡通 / 游戏感 | 加引"vintage cartography, hand-drawn campaign map, before-printing-press era" |
| 画了完整地图而非一角 | prompt 强调 "showing only a CORNER of a larger map; the rest is implied to be hidden under the rolled-up sides" |
| 羊皮卷外有桌面 / 阴影 | prompt 强调 "background is plain white; do NOT render desk, shadow, or any other object" |
