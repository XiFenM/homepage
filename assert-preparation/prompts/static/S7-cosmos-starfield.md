# S7 · 观星塔星空背景 (Cosmos Starfield)

> **Wave 3** · 工具 = gpt-image-2 · 关联组件 [DESIGN.md §5 Cosmos Starfield](../../../DESIGN.md)

## 1. 用途与规格 (Spec)

| 项 | 值 |
|---|---|
| 出现位置 | 观星塔 hotspot 触发后的全屏接管背景（探索站从羊皮卷过渡到星空） |
| **本次输出** | **3840×2160 PNG**（4K UHD，16:9，API 最大边长限制） |
| 后处理 | WebP 转码（80% 质量），归档至 `public/art/cosmos/starfield.webp` |
| Token 锚点 | `cosmos-deep-night` = `oklch(15% 0.03 260)` (底) · `cosmos-pearl-blue` = `oklch(72% 0.08 240)` (银河带与星座线) · `cosmos-ice-white` = `oklch(95% 0.01 240)` (星点) |
| 风险等级 | **中** — 星点分布自然性 + 不滑向"NASA 写实星云"的克制感 |

## 2. 主 Prompt (中文主版)

```
生成一张深邃宁静的夜空油画式数字插画。整张画面是一片观星塔上方的天空——访客的视点是站在观星塔顶部圆穹天文台望出去，所以画面是向上看的天空、不是地面景观。

【底色】
- 主底色 OKLCH 15% 0.03 260（深邃的夜蓝紫色——**不是纯黑、不是 #000**，要有一丝紫蓝偏移）
- 整张画面的"黑色"都通过这个深夜蓝紫色实现，绝对禁止纯黑

【星点分布——必须严格按三层深度绘制】

**远层（深度最远，最暗最小）**：
- 约 200 颗微小星点（每颗约 1px 等效，在 4K 下约 2–3 像素），颜色 OKLCH 95% 0.01 240（cosmos-ice-white，蓝偏移的白）
- 整体亮度低（明度感约 30–40%）
- **自然散布**——不要呈网格状、不要均匀间距，应有疏密变化，像真实夜空

**中层（深度中等）**：
- 约 80 颗中等星点（每颗约 2px 等效，在 4K 下约 4–6 像素），颜色 OKLCH 95% 0.01 240
- 亮度中等（明度感约 60–70%）
- 自然散布

**近层（深度最近，最亮最大）**：
- 约 30 颗较大星点（每颗约 3–5px 等效，在 4K 下约 8–14 像素），颜色 OKLCH 95% 0.01 240，**带柔和的径向光晕**（光晕颜色 OKLCH 72% 0.08 240 cosmos-pearl-blue，光晕直径约星点的 2–3 倍）
- 亮度高（明度感 90–100%）
- 这些是"主角星"——但仍然散布自然，不集中、不堆叠

【银河带】
- 一道**柔和的银河带**斜向横穿画面（左下到右上，或左上到右下）
- 银河带是一片**模糊的云状色块**，颜色 OKLCH 72% 0.08 240（pearl-blue），整体不透明度约 30%
- 银河带边缘**柔和扩散**（不是硬边），带状宽度约画面短边的 30–40%
- 银河带本身也散布着更密集的微小星点（远层星增多于此处）

【星座连线】
画面中**两个识别度高的星座**用极细线（1–2 像素）连接，颜色 OKLCH 72% 0.08 240，不透明度约 60%：
- **猎户腰带（Orion's Belt）**：3 颗大致等距的明亮星点（属近层），用极细线连接成短直线
- **仙后座 W（Cassiopeia）**：5 颗明亮星点（属近层）以 W / M 形排列，用极细线连接

【完全禁止——克制感至关重要】
- **不要画行星（地球、火星、土星等）**
- **不要画月亮**
- **不要画高饱和度星云**——任何彩色发光气体云、任何 NASA 哈勃望远镜风格的玫瑰色 / 翡翠绿 / 宝蓝色 / 金色星云
- **不要画流星 / 彗星**
- **不要画极光**
- **不要画地球大气层 / 云**
- **不要任何文字、字母、星座名标注、坐标**
- **不要发光特效 / scifi 元素**

【整体氛围】
- 这是一个**安静的天文台在晴朗夜晚的视野**，不是宇宙事件、不是科幻太空、不是星图教科书
- 像 Studio Ghibli 在《魔女宅急便》《侧耳倾听》中的夜空、新海诚《追逐繁星的孩子》早期场景的克制夜空
- 整体感觉应当**清冷、深邃、安静、足以让访客屏住一秒呼吸**

【绘画风格】
- 极简油画风 + 极轻的胶片颗粒 grain（模拟胶片夜景摄影的质感，但不是真实摄影）
- 笔触柔和但不模糊
- **不是 NASA 哈勃照片**、**不是 Astronomy Picture of the Day** 视觉风、**不是科幻游戏 splash screen**
- 灵感参照：复古星图（19 世纪 atlas 风）、Studio Ghibli 夜空背景、京都动画《CLANNAD》黄昏星空、天文台官网朴素海报

输出：3840×2160（4K UHD，16:9 横向），无文字、无水印、无任何边框装饰，整张画面满铺。
```

## 5. 验收清单 (Acceptance Checklist)

### 5.1 通用项

- [ ] 底色 OKLCH 抽 3 点落在 15%±5% L、0.03±0.02 C、260±15° H
- [ ] **不出现纯黑** OKLCH L < 10% 区域
- [ ] 银河带 / 星座线色 OKLCH 抽 1 点落在 72%±5% L、0.08±0.02 C、240±15° H
- [ ] 无 anti-reference 触发
- [ ] **完全无文字 / 字母 / 数字 / 标注**

### 5.2 星点分布项

- [ ] 三层星点都存在（远小暗 / 中等 / 近大亮）
- [ ] 总星点数量大致符合 200+80+30 的比例感（不要稀疏到只有几十颗）
- [ ] 星点**自然散布**——无网格状、无均匀间距、有疏密变化
- [ ] 近层星点有柔和径向光晕

### 5.3 银河带与星座

- [ ] 一道斜向银河带存在，柔和扩散
- [ ] 银河带带状宽度约画面短边 30–40%
- [ ] 出现至少 1 个识别度高的星座（猎户腰带或仙后座 W）
- [ ] 星座连线极细（1–2px）、不喧宾夺主

### 5.4 克制感（最重要）

- [ ] **没有彩色发光星云**（玫瑰色、宝蓝、翡翠等）
- [ ] **没有行星 / 月亮 / 流星 / 极光**
- [ ] **没有地球大气层 / 云**
- [ ] 整体氛围"安静的天文台之夜"而非"宇宙史诗"
- [ ] 油画手绘感，不是 NASA 写实摄影

## 6. 已知风险与回退

| 失败模式 | 应对 |
|---|---|
| 出现高饱和星云 | prompt 反例段加粗 "ABSOLUTELY NO COLORFUL NEBULAE, NO HUBBLE-STYLE GAS CLOUDS, NO NASA APOD AESTHETICS" |
| 星点呈网格状 | "stars distributed organically and irregularly, NOT in any grid or pattern, with natural density variation" |
| 出现行星 / 月亮 | "ONLY stars, Milky Way band, and constellation lines. NO planets, NO moon, NO any other celestial bodies." |
| 整体偏黑死板 | "deep night blue with violet undertone, NOT pure black, the void itself has subtle color depth" |
| 出现星座名 / 坐标 | gpt-image-2 文字问题，prompt 顶部加粗 "NO TEXT, NO LETTERS, NO NUMBERS ANYWHERE" |
| 星座连线过粗 | "thin hairline connecting lines, 1–2 pixel equivalent, low opacity 60%, almost imperceptible" |
