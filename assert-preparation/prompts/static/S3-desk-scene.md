# S3 · 首页书桌场景（无物件）(Desk Scene Background)

> **Wave 2** · 工具 = gpt-image-2 · 关联组件 [DESIGN.md §5 Homepage Fork Scene](../../../DESIGN.md)

## 1. 用途与规格 (Spec)

| 项 | 值 |
|---|---|
| 出现位置 | 首页 hero 全屏背景 |
| **本次输出** | 3840×2160 PNG（4K UHD，16:9） |
| 后处理 | WebP 转码（80% 质量），归档至 `public/art/homepage/desk-scene.webp` |
| Token 锚点 | `desk-walnut` = `oklch(28% 0.04 55)` · `desk-walnut-deep` = `oklch(20% 0.03 50)` · `lamp-glow` = `oklch(82% 0.10 75)` · `dust-mote` = `oklch(70% 0.02 60)` |
| 合成关系 | S4 羊皮卷将合成至**左中**、S5 春绿便签将合成至**右中**——本场景需为这两个物件留出可呼吸的空位 |

> **API 注意**：3840×2160 是 gpt-image-2 文档明示的"experimental，可能不稳"分辨率上限。第一轮用 4K 跑；若 4 张全废，回退到 2560×1440 重跑。

## 2. 主 Prompt (中文主版)

```
生成一张俯视角（约 75° 倾斜）的暖灯下书桌油画式数字插画。视角像是从天花板斜向下看一张老书桌——你能完整看到桌面，桌面边缘呈等距透视。

桌面材质：温暖的胡桃木 (OKLCH 28% 0.04 55)，可见手绘的木纹笔触（不是 3D 渲染的木纹贴图）。桌面整体偏暖偏冷的深木色——胡桃木的微妙冷光，不是橡木的偏黄、也不是樱桃木的偏红。

灯光：画面左上方有一盏温暖的台灯（不需要把灯本体画完整，只要画出它在桌面上投下的圆形暖光圈即可）。光圈中心 OKLCH 82% 0.10 75，边缘衰减到桌面本色。光圈直径约占画面宽度的 60%，居中略偏左上。光圈内桌面被暖光照亮，光圈外桌面渐渐暗下去到 OKLCH 20% 0.03 50 的深胡桃木阴影。

飘尘：在光圈范围内（特别是光线侧斜处），可见 4–6 颗微小的尘埃颗粒（OKLCH 70% 0.02 60，约 4–6 像素大小，半透明 30%–60% opacity 不一），像是被光打亮的悬浮尘埃。

【重要：桌面必须完全空着。无纸张，无物件，无书本，无杯子，无文字，无任何字母或符号。】桌面中央与左右两侧均需保留充足的负空间——稍后会有两件物品（一卷羊皮地图、一张春绿便签）通过 HTML 合成在桌面上，分别落在左中和右中。所以构图必须给"左中"和"右中"两块各留出约画面宽度 25% × 高度 35% 的空白桌面区。

背景（桌面以外）：远处隐约可见书房的内部——书架的剪影逐渐沉入黑暗，不要详细的物件，只要"夜深、屋内、独处"的氛围。书架剪影的颜色比桌面更深一档，用 OKLCH 18% 左右的色值带过，避免抢戏。

整体氛围：深夜、安静、像桌游夜聚会结束后的暖灯桌——一种"刚才有人在这里、现在屋里只有灯"的留白感。

材质与笔触：能看见画家的笔触，特别是木纹与光圈过渡处。微弱的纸面颗粒感（grain）覆盖整张图，强度约 8%。

风格参考：Studio Ghibli（吉卜力）室内背景画、新海诚《秒速五厘米》室内夜景、TRIGGER 早期作品的室内背景、《迷宫饭》中厨房与小屋的暖光场景。

反例：NOT 照片真实感，NOT 3D 渲染，NOT 摄影作品，NOT 赛博朋克，NOT 工业风，NOT 现代极简风，NOT 高饱和度，NOT 商业产品广告画面，NOT 写实皮肤光影，NOT 游戏 CG，NOT 电脑桌面壁纸感，NOT 任何屏幕 / 显示器 / 键盘 / 鼠标 / 电源线等现代办公元素出现。

输出：3840×2160（4K UHD，16:9 横向），无文字、无水印、无装饰边框。
```

## 5. 验收清单 (Acceptance Checklist)

### 5.1 通用项

- [ ] 桌面色 OKLCH 抽 3 点落在 28%±5% L、0.04±0.02 C、55±10° H
- [ ] 灯光圈中心色 OKLCH 抽 1 点落在 82%±5% L、0.10±0.02 C、75±10° H
- [ ] 桌面阴影色 OKLCH 抽 1 点落在 20%±5% L 附近
- [ ] 无 anti-reference 触发

### 5.2 场景特定项

- [ ] **俯视角约 75°**——能看到完整桌面、桌面边缘有等距透视
- [ ] **桌面完全空**——无纸张、物件、书本、杯子、文字、键鼠等任何元素
- [ ] **左中、右中各有约 25%×35% 的可合成空位**——既不被光圈中心遮挡也不被阴影吞掉
- [ ] **灯光圈居中略偏左上**，覆盖画面 60% 宽度，从中心暖到边缘衰减自然
- [ ] **飘尘 4–6 颗**——在光圈侧斜处可辨，不是噪点也不是雪花
- [ ] **背景书架剪影存在但不喧宾夺主**——只是夜屋氛围背景
- [ ] **油画式笔触可见**——不是照片、不是 3D
- [ ] **整体偏冷的暖**——温暖但不黄不橘，桌游夜的灯感而非烛光感

## 6. 已知风险与回退

| 失败模式 | 应对 |
|---|---|
| 桌面有意外物件（笔记本、杯子、笔等） | prompt 反例段加粗"桌面完全空"；如仍出现，先 generate 再 inpaint 抹掉 |
| 视角不是俯视而是平视/正面 | prompt 加强"camera positioned above the desk looking straight down at 75°" |
| 4K 出图崩坏（怪异纹理、模糊） | 回退 2560×1440，再 PIL/Photoshop 上采样 |
| 光圈太集中或太分散 | 改"60% 宽度"为更精确的"radius covers 30% of image width from center" |
| 风格偏写实摄影 | 加粗"painterly digital illustration, NOT photorealistic"；引用更具体的 Ghibli 室内场景片段 |
